# 六问法教练训练

这组文件的目标很简单：

- 先帮你吃透六问法本体
- 先把“六问法教练”该怎么带人想清楚
- 先把第三问和第四问的边界彻底分开
- 暂时不把重点放在 skill 设计上

今天这组材料，优先回答的是：

1. 六问法到底在解决什么问题
2. 六问法六问之间的逻辑关系是什么
3. 第三问和第四问为什么不能混
4. 作为教练，应该怎么带，而不是怎么替用户答
5. 达到“能带人做六问法”的最低标准是什么

## 你学完后，至少应该能做到

- 能判断一个问题适不适合用六问法
- 能把目标、现状、差距放在同一维度上
- 能在第三问只做“差距界定”，不提前做第四问
- 能在第四问识别“目标”到底是真目标，还是临时方案
- 能在第五问用框架和 Why 去分析原因
- 能在第六问把结果收束成“真正的问题”
- 能发现什么时候应该重开
