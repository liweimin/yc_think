# Examples

## Fit example

User:

- Use yc_think. I keep postponing my monthly learning tasks and I am not sure what the real problem is.

Expected pattern:

1. classify as a fit
2. identify likely domain as personal execution or habit
3. respond in Chinese by default
4. do not finish the whole analysis immediately
5. explain the first step and ask the next question first
6. only give answer directions if the user asks for them or gets stuck
7. use 0.5Why only in question 4 and question 5 when those stages arrive
8. possibly separate mindset causes from execution causes
9. produce the final real problem only after enough interaction or if the user asks for a draft

## Not-fit example

User:

- Use yc_think. What does SMART goal mean?

Expected pattern:

1. classify as a pure information question
2. explain that six questions is not the right tool
3. answer directly
4. stop without forcing the full workflow

## Restart example

User:

- Use yc_think. I want to quit my job this month.

Expected pattern:

1. begin gating and likely classify as a choice or provisional-solution statement
2. if running into six questions, question 4 may show the real issue is not quitting itself
3. restart from the deeper goal if needed
4. preserve the original problem and the restart reason

## Draft-mode example

User:

- Use yc_think. 先别互动，直接按现有信息给我一版初步分析，后面我再修正。

Expected pattern:

1. say that this is a provisional draft
2. mark assumptions clearly
3. keep the six-question structure
4. invite correction after the draft
