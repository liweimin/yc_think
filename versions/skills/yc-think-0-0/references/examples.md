# Examples

## Fit example

User:

- Use yc_think. I keep postponing my monthly learning tasks and I am not sure what the real problem is.

Expected pattern:

1. classify as a fit
2. identify likely domain as personal execution or habit
3. run six questions
4. use 0.5Why in question 4 and question 5
5. possibly separate mindset causes from execution causes
6. produce the final real problem, constraints, changeable factors, and first-pass directions

## Not-fit example

User:

- Use yc_think. What does SMART goal mean?

Expected pattern:

1. classify as a pure information question
2. explain that six questions is not the right tool
3. answer directly
4. stop without forcing the full workflow

## Restart example

User:

- Use yc_think. I want to quit my job this month.

Expected pattern:

1. begin gating and likely classify as a choice or provisional-solution statement
2. if running into six questions, question 4 may show the real issue is not quitting itself
3. restart from the deeper goal if needed
4. preserve the original problem and the restart reason
